Key Facts

  • Fore is a set of HTML Web Components.
  • Architecture is inspired by XForms 2.0
  • When the Fore import gets loaded the Custom Elements are defined and their behavior gets attached.
  • Fore will start its lifecycle by setting up a model.
  • fx-output outputs a bound value
  • If there is no model one will be created using the ref attributes to create data nodes.
