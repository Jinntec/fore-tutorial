Key Facts

  • a Fore page can have as many instances as necessary
  • The first instance in document order is the default instance. If no id is specified the id will be 'default'
  • All but the first instance in document order need to specify an id
  • The datatype of the instance defaults to 'XML'
  • The insert action allows to insert nodes into the instance data
  • As a template for insertion the origin attribute can be used.
